IMDb
Titles - Movies
Principals - Actors

Roles: Anonymous User
       Editor
	   Publisher
CRUD: POST, GET, PUT, DELETE

- As an anonymous user, I want to be able to
  get a list of all movies, so that I can check
  which movies I want to watch next.
  
  GET /movies
  
- As an anonymous user, I want to be able to get
  the details about a specific movie by it's unique id
  so that I can see if it's something I want to 
  watch.
  
  GET /movies/{movieid}   // movieid = tt3704428, tt1745960
  
- As an anonymous user, I want to be able to search
  for a movie based on part of a movies name so that
  I can find the movie I want to watch.
  
  GET /movies?name=Hobbit
  
- As an anonymous user, I want to be able to see
  a list of all actors/actresses that are in a particular movie.
- As an anonymous user, I want to be able to view all movies
  for a particular actor/actress.
- As an anoymous user, I want to be able to search for
  an actor/actress based on all or part of their name so
  that I can filter the list quickly.
- As an publisher, I want to be able to create movies
  so that I can add new information for users to view.
- As an editor, I want to be able to change, correct or
  add information about a movie so that the information is
  current and accurate.  
- As a publisher, I want to be able to delete movies so
  that I can remove invalid or incorrect information.
